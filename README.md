## CLIP
自然言語教師から転送可能な視覚モデルの学習<br>
GitHubで公開されているコードを動かした記録<br>
```$ pip install git+https://github.com/openai/CLIP.git```<br>
として，ライブラリが用意されているため，Cloneする必要もなく，簡単に実装可能．<br>
